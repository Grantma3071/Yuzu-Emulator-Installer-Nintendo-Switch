## 🚀 How to Use
1. Extract archive  
2. Run yuzu.exe or yuzu-qt.exe as Administrator  
3. Add your prod.keys & title.keys (legally dumped)  
4. Add game folder or install .nsp/.xci files  
5. Configure graphics & controls  
6. Right-click game → Play  
Works on Windows 10/11.
